## Development

### Requirements

* [Node JS](https://nodejs.org/) (>= 10)
* [Yarn](https://yarnpkg.com/) (>= 1.10.0)
* [RNA cli](https://www.npmjs.com/package/@chialab/rna-cli)

### Build the project.

```sh
$ yarn install
$ yarn build
```

### Develpment mode.
```sh
$ yarn watch
```
